Internet Banking

Beaver Bobby is sitting in a cafe when he suddenly remembers his friend Dotty paid for his ice-cream last Sunday, and he still owes her. Luckily, the cafe has WiFi and Bobby has his laptop computer with him, so he decides to settle his debt at once, before he forgets again.

Which of the following should Bobby type to the address bar of his web browser to securely connect to his bank (assuming the bank's web server is called www.bank.com)?

  1. http://www.bank.com
  2. https://www.bank.com
  3. www.bank.com
  4. bank.com

The correct answer is: B

The correct answer is B (https://www.bank.com), because only in this case the web browser starts an encrypted and authenticated session at once.

In case A (http://www.bank.com), the web browser starts an unencrypted and unauthenticated connection first. The bank's server would most likely redirect the browser to the secure connection, but in a public wireless network it is rather easy for an attacker to intercept the initial connection and redirect it anywhere (for example, to a phishing site) before it ever gets to the real bank's server.

The answer C is really the same as A, as most web browsers default to plain HTTP, not the encrypted and authenticated HTTPS, if the protocol is not given explicitly. The name in answer D may or may not be an alias for the same server named in the other answers, but in any case, the browser most likely defaults to plain HTTP again.

It's Informatics!

It is important to know (and follow!) secure procedures to avoid becoming a victim of identity theft.
