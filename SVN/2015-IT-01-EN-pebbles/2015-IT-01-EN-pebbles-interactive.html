Pebble game

The goal of this puzzle game is to put all the red pebbles in the first box by following one simple rule: a box can be emptied only if the number of pebbles in it is equal to the number of the box itself; when a box is emptied, its pebbles are distributed equally on all the boxes to its left.

Click on pebbles to empty the corresponding box.

Question

Is this puzzle solvable? If yes, give the order of the boxes to follow in emptying the boxes.

Answer

A puzzle is solvable only when the winning strategy is applicable until the end. The winning strategy chooses always the lowest index box which can be emptied. In fact, when two boxes can be chosen, if one empties the highest one, the lower will be blocked forever.

Alternative question

Give a solvable puzzle with a total of 5 pebbles.

Answer

1,1,3,0,0,0,0

Alternative question

Which is the maximum number of pebbles for a solvable puzzle?

Answer

There are only 21 (22 if you consider the trivial one) solvable puzzles. The maximum number of pebbles in solvable puzzles is 21.
